
# Fresh OS Setup 

This repository contains my personal dotfiles, scripts, and setup configurations for fresh installations of Linux and Windows. The goal is to quickly bootstrap a clean OS with my preferred settings.

## Features
- Configurations for Bash, PowerShell, Vim, Git, and more.
- Scripts to automate package installation and system setup.
- Custom aliases, functions, and tweaks.
